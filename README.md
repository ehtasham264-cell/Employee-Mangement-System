# Employee Management System

## Description
A web-based Employee Management System built using Django. It allows users to manage employee records.

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee List
- Search Employee
- Login System

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
git clone https://github.com/ehtasham264-cell/Employee-Management-System.git
cd Employee-Management-System
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
